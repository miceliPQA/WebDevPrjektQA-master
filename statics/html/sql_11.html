CHEATSHEET - SQL

Definition: SQL-Cheatsheet

  • Datenabfrage: Mit SQL können Benutzer Daten aus einer oder mehreren Tabellen abrufen, indem sie SELECT-Abfragen verwenden. Sie können Bedingungen festlegen, um die abgerufenen Daten zu filtern und die Ergebnisse nach verschiedenen Kriterien zu sortieren.
  • Datenmanipulation: SQL bietet Möglichkeiten zur Einfügung (INSERT), Aktualisierung (UPDATE) und Löschung (DELETE) von Daten in einer Datenbank. Dadurch können Benutzer neue Datensätze hinzufügen, vorhandene aktualisieren oder nicht mehr benötigte Datensätze entfernen.
  • Datenbankstrukturverwaltung: Mit SQL können Benutzer Datenbanken, Tabellen, Indizes und andere Objekte erstellen, ändern oder löschen. Sie können Tabellen erstellen, Spalten hinzufügen oder entfernen, Indizes definieren und Einschränkungen festlegen, um die Datenintegrität zu gewährleisten.
  • Datenbankadministration: SQL bietet auch Befehle für die Benutzerverwaltung, Zugriffssteuerung und Sicherheit in einer Datenbank. Benutzer können Benutzerkonten erstellen, Berechtigungen vergeben und den Zugriff auf Daten und Datenbankobjekte steuern.

Beispiel-Aufgabe 1:

Lösung 1 mit EquiJoin:

  • SELECT P.Datum_Soll AS Datum, WEEKDAY(P.Datum_Soll) AS Wochentag, T.Bezeichnung AS Tätigkeit
  • FROM Pflegearbeiten AS P, Objekt AS O, Tätigkeit AS T Where O.OID = P.OID AND T.TID = P.TID AND Datum BETWEEN '2023-06-19' AND '2023-06-30' AND O.Bezeichnung = 'Außenlage Nord'
  • ORDER BY Datum ASC;


    • Lösung 2 mit InnerJoin:

  • SELECT P.Datum_Soll AS Datum, WEEKDAY(P.Datum_Soll) AS Wochentag, T.Bezeichnung AS Tätigkeit
  • FROM Pflegearbeiten AS P INNER JOIN Objekt AS O ON O.OID = P.OID INNER JOIN Tätigkeit AS T ON T.TID = P.TID
  • WHERE P.Datum_Soll BETWEEN '2023-06-19' AND '2023-06-30' AND O.Bezeichnung = 'Außenanlage Nord'
  • ORDER BY Datum ASC;
