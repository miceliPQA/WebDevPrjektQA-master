Lernprojekt U13 - QualifizierungsAkademie RheinRuhr GmbH & CO.KG.

Lernprojekt U13

Ausbildungsbetrieb: QualifizierungsAkademie RheinRuhr GmbH & CO.KG.

Ausbilder: Philipp Braun

Inhalte

  • HTML
  • CSS
  • JavaScript
  • Strukturierte Programmierung in Form von Programmierungsaufgaben, die mit Struktogramm, PAP oder Pseudocode gelöst werden müssen
  • UML-Aktivitäts- und Sequenzdiagramme, die ebenfalls mithilfe von IHK-Aufgaben gelöst wurden
  • Implementierung des Frontend-Codes in ein Django-basiertes Backend
  • Praktische Vermittlung des MVT (Model-View-Template) bzw. MVC (Model-View-Controller)-Patterns mit UML-Diagrammen

Das Lernprojekt U13 bei der QualifizierungsAkademie RheinRuhr GmbH & CO.KG. umfasst eine Vielzahl von Inhalten. In den Bereichen HTML, CSS und JavaScript lernen die Teilnehmer grundlegende Webtechnologien. Sie setzen ihr Wissen in strukturierten Programmieraufgaben um, bei denen sie mithilfe von Struktogrammen, PAP (Programmablaufplänen) oder Pseudocode Lösungen erarbeiten.

Zusätzlich beschäftigen sich die Teilnehmer mit UML (Unified Modeling Language) und erstellen Aktivitäts- und Sequenzdiagramme, wobei sie praktische Aufgabenstellungen der IHK bearbeiten. Im Anschluss wird der Frontend-Code in ein Django-basiertes Backend implementiert. Dabei werden das MVT (Model-View-Template) bzw. MVC (Model-View-Controller)-Pattern vermittelt und mit Hilfe von UML-Diagrammen visualisiert.
